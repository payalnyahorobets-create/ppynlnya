# ppynlnya

Локальний веб-додаток для імпорту CSV та розрахунків за логікою таблиці аналізу.

## Запуск

1. Запустіть локальний сервер у корені репозиторію:

```bash
python -m http.server 8000
```

2. Відкрийте браузер та перейдіть на `http://localhost:8000`.

## Дані для імпорту

Підтримуються файли CSV, XLSX та XML.

* **Номенклатура**: Артикул, Код, Ім'я, Категорія, Ціна закупівлі, Ціна продажу, Кількість, Штрихкод.
* **ID категорій**: Категорія, ID.
* **Перші продажі**: SKU, Перша_продаж.
* **ShelfDates**: Артикул, Дата_полки.
* **Продажі**: SKU/Артикул, Кількість, Виторг.

Імпортовані дані зберігаються локально у браузері (localStorage).

## Розділи

* **Товари**: номенклатура, залишки по філіях, фільтри по категоріях.
* **Характеристики**: редагування атрибутів товарів по категоріях.
* **Закупки**: огляд залишків для формування переміщень та закупівель.
* **Продажі**: помісячна статистика та перегляд продажів.
* **Аналітика**: ABC-XYZ, оборотність, маржинальність.
